Introduction

What is Gaen?

A cross platform, {{=A('component based game engine', _href='http://en.wikipedia.org/wiki/Entity_component_system')}} that automates CPU concurrency.

Game designers create components and entities and designate data dependencies. The engine load balances these entities across all available cores. Running on a machine with more cores will automatically improve performance without any code modifications.
